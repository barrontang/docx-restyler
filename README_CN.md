# docx-restyler

[English](README.md)

`docx-restyler` 是一个面向 OpenClaw 的 Word 文档重排版 skill，用来把一个**源 Word 文档**按**参考模板 Word 文档**的样式重新排版，输出新的定稿版 `.docx`。

## 它解决什么问题

很多真实办公场景里，用户已经有：
- 一份标准模板 Word 文件
- 一份内容完整但排版不规范的源 Word 文件

目标不是改写内容，而是：

**让源文档按模板样式重新定稿。**

这类需求在下面场景里很常见：
- 正式报告
- 汇报材料
- 企业内部文档
- 中文公文/正式材料
- 模板统一化定稿

## 当前思路

当前原型重点是：
- 学习模板的样式规则
- 保留源文档内容
- 输出新的 `.docx`

第一版主要关注：
- 标题
- 正文段落
- 基础表格
- 尾部署名/日期
- 中文正式文档的层级识别

## 特别处理

这个项目对中文正式材料的标题层级做了专门规则支持，例如：
- `一、`
- `（一）`
- `1.` / `1、`
- `（1）`

因为很多真实 Word 文档并不会规范使用内建样式名称，所以不能只依赖 Word 自带的 `Heading 1 / Heading 2`。

## 当前限制

当前仍然是原型阶段，限制包括：
- 复杂表格样式还不够稳定
- 页眉页脚等高级对象暂未重点处理
- 复杂 Word 结构不能保证完全保真
- 更精细的排版控制仍可继续增强

## 一句话总结

**`docx-restyler` 是一个 OpenClaw skill 原型，用来把源 Word 文档按参考模板的样式重新排版，特别针对中文正式文稿的标题层级做了优化。**
