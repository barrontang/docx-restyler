# docx-restyler

[English](README.md)

**一个面向 OpenClaw 的 Word 文档定稿 Skill 原型。**

给它：
- 一份标准模板 `.docx`
- 一份源文档 `.docx`

它会输出：
- 一份按模板样式重新排版后的定稿版 `.docx`

## 它解决的问题

很多真实办公场景里，最耗时间的不是“写完文档”，而是：

- 调标题
- 调正文
- 调字号
- 调行距
- 调段前段后
- 套模板
- 做最终定稿

`docx-restyler` 想解决的，就是这部分重复、机械、但又高频的工作。

## 当前能力

当前版本重点支持：
- 标题层级识别
- 正文样式重排
- 基础表格保留
- 尾部署名/日期处理
- 中文正式文稿层级规则识别

特别针对这类层级：
- `一、`
- `（一）`
- `1.` / `1、`
- `（1）`

## 适合场景

- 正式报告
- 企业内部汇报
- 中文正式材料
- 模板统一化定稿

## 当前状态

这是一个已经能处理真实文档的原型，但还不是最终版。

它更适合被理解为：
- 一个正在迭代的文档定稿引擎
- 一个面向 OpenClaw 的 Skill 原型
- 一个可以在真实文档中不断校正的工具

## 一句话总结

**`docx-restyler` 是一个 OpenClaw skill 原型，用来把源 Word 文档按参考模板的样式重新排版，特别针对中文正式文稿的标题层级做了优化。**
