A lot of document work isn’t writing.
It’s reformatting.

I built **docx-restyler** to help with exactly that:
- source `.docx`
- template `.docx`
- finalized `.docx`

It’s now an OpenClaw skill prototype, with extra handling for Chinese formal heading structures like:
- 一、
- （一）
- 1.
- （1）

Repo:
https://github.com/barrontang/docx-restyler
