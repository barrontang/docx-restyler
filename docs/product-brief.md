# docx-restyler product brief

## Positioning

docx-restyler is an OpenClaw skill prototype for template-driven Word document finalization.

## Core problem

Users often already have:
- a standard `.docx` template
- a source `.docx` containing the actual content

The pain point is repetitive manual formatting before final delivery.

## Core solution

Use the template document as the formatting reference and the source document as the content source, then generate a new finalized `.docx`.

## Target scenarios

- internal reports
- official briefings
- structured enterprise documents
- Chinese formal/government-style materials

## Product principles

- preserve content
- normalize formatting
- prioritize heading/body correctness
- iterate using real documents instead of synthetic demos only

## Current status

Prototype, tested on real documents, acceptable for controlled iteration, not yet a finished production-grade formatter.
